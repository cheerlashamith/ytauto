AUTOCOURSE FULL ARCHITECTURE AND RUNBOOK
========================================

SECTION 1: WHAT I UNDERSTOOD
----------------------------

You need a final system with these modes:

1. Manual Course Mode
   User gives a topic or script.
   System generates a course video plan.
   Manim renders serious course animations.
   MoneyPrinterTurbo renders voice/subtitles/final MP4.

2. Story Mode
   User gives a kids story/moral story topic.
   Story generator creates professional story scenes.
   ComfyUI creates cartoon/story visuals.
   MoneyPrinterTurbo renders voice/subtitles/final MP4.

3. YouTube Extraction Mode
   User gives a YouTube URL.
   System extracts transcript in future.
   AI rewrites script and scene plan.
   Visual router chooses Manim/ComfyUI.
   MoneyPrinterTurbo renders final MP4.

4. Autonomous Mode
   AI thinks of a topic or uses syllabus/trends.
   AI creates script and scene plan.
   Visual router chooses renderer.
   MoneyPrinterTurbo renders final MP4.

5. Syllabus-Aware DSA Mode
   System has JNTUK R23 Data Structures syllabus.
   Agent knows units, topics, subtopics, lab exercises.
   If user asks broad topic, agent asks clarification.
   If user asks exact topic, agent generates course plan.

SECTION 6: RUN STORY MODE
-------------------------

Use for kids stories and cartoon storytelling.

1. Go to:

    cd /d C:\DOWNLOAD_ME_AutoCourse_Final_Master\02_engines\story_mode_professional_rewrite

2. Generate professional story JSON:

    python -m story_mode.generate_story --topic "A fox learns kindness" --out scenes_story.json --style storybook

3. Use existing ComfyUI story material generator from your working story_mode folder, or connect this JSON to ComfyUI pipeline.

4. Required ComfyUI workflow location in active story engine:

    workflows\sd3.5_simple_example_api.json

5. Output clips go to generated\clips.

6. In MoneyPrinterTurbo:

    Video Source = Local file
    Upload generated clips
    Paste script
    Paste keywords
    TTS = Azure TTS V1
    BGM = 0.08
    Aspect = Portrait 9:16


SECTION 7: RUN MANIM COURSE MODE
--------------------------------

Use for serious course topics.

1. Go to:

    cd /d C:\DOWNLOAD_ME_AutoCourse_Final_Master\02_engines\manim_ultimate_course_renderer

2. Install requirements:

    python -m pip install -r requirements.txt

3. Check Manim:

    python -m manim --version

4. Put your course JSON here:

    scenes_course.json

5. You can use sample:

    copy samples\electricity_advanced.json scenes_course.json

6. Render:

    render_all.bat

7. Output clips:

    generated\clips

8. Script and keywords:

    generated\moneyprinter_video_script.txt
    generated\moneyprinter_keywords.txt

9. In MoneyPrinterTurbo:

    Video Source = Local file
    Upload generated clips
    Paste script and keywords
    Transition = None
    Aspect = Landscape 16:9
    Max Clip Duration = 10
    TTS = Azure TTS V1
    BGM = 0.05
