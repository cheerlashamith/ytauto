# Modes Guide

## Manual Course Mode
User enters a topic or script. Backend creates course plan. Use Manim for visual clips.

## Story Mode
User enters story topic. Backend creates story plan. Use ComfyUI for cartoon clips.

## YouTube Extraction Mode
User enters URL. Current version creates a plan placeholder. Future version will use yt-dlp/transcript extraction.

## Autonomous Mode
User enters broad goal. Backend creates a topic/plan placeholder. Future version will use trend/research agent.
